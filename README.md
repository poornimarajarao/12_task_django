# A Simple Django Project Using Core Concepts

## Concepts Used

* **Rendering HTML Templates:** Using function-based views to display content dynamically.
* **URL Mapping :** Django maps URLs to views in `urls.py` to determine which function handles each request.
* **URL Navigation:** Users can move between pages (Food and Instamart) using links or buttons in templates.
* **Managing Static Files:** Handling images, CSS, and JS using `{% load static %}`.
* **Template Inheritance:** Reusing layout and structure from a parent template using `{% extends %}` and `{% block %}`.
## Tech Stack 

* **Backend Framework:** Django (Python)
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Template Engine:** Django Templates (`{% extends %}`, `{% block %}`, `{% load static %}`)
* **Static Files Management:** Django static files for images, CSS, and JS
* **Server:** Django development server (`python manage.py runserver`)
